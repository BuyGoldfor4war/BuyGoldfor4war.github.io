I need to clarify that this is not the official Burger King website, it is a small demonstration of my skills through a web page redesign.

I have taken care of every detail when making this website, counting on a particular color palette using all the wide range of possibilities that javascript offers.

I started by making a "reference" in figma in order to have a base on which to work, above all the most important thing for me in a website is how easy and simple it looks, in this case for the initial carousel I decided to leave the navigation buttons below to give it more of an mp3 vibe

When I work, I always try to prioritize dynamism, so as I program each element, I also check that it works properly on each device and looks stylized. In this way I can speed up my productivity.
